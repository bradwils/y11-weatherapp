Enter the place in which you would like to receive weather information from in the search bar (at the top of the page)

Please note that not all areas are supported, so if you continuosly get error messages for a location, please try another location instead.


Toggle between the map and a recent search tab using the button available on the right of the page that is named "Toggle Map/Recent"


The weather information of the place will appear in the main area of the website page along with weather animations

If there are no weather animations avaliable for that scenario, then it will leave a note underneath the weather result's text.


If you would like to force preview a specific scenario, please enter 'override ' + (weather condition) to force preview it. For example, 'override rain' Please ensure that it is in all lowercase.
The provided text description will not line up with the provided weather scenario.

The available weather scenarios are rain, thunderstorm, snow, clear, drizzle, and clouds.
